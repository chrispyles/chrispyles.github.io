Work Samples

As quick sample of the kind of work that goes on in UC Berkeley introductory CS classes, here is one of the homework problems I completed in CS 88: it is a function that computes the hailstone sequence of a number n and returns the number of steps in that sequence. A hailstone sequence goes like this:

While n is not 1:

1. If n is even, divide it by 2.

2. If n is odd, multiply it by 3 and add 1.

3. Repeat until n is 1.

Here is the program I wrote to find the hailstone sequence of a number n (note that the code follows the same sequence of actions as the description above; a good strategy to follow when writing a program is to follow explicit, step-by-step instructions until the desired result is achieved):

def hailstone(n):
print(n)
steps = 1
while n != 1:
if n % 2 == 0:
n = int(n/2)
print(n)
steps += 1
else:
n = int(3*n+1)
print(n)
steps += 1
return steps


If you run hailstone(10), you would get this output:

>>> a = hailstone(10)
10
5
16
8
4
2
1
>>> a
7
